# 🛠️ Minecraft Android Install Fixer

Простое и эффективное приложение для решения проблем с установкой и обновлением Minecraft на Android. 

### 🚀 Что оно умеет?
- **Root Mode:** Полная очистка системных папок игры и кэша Google Play (для тех, у кого есть права суперпользователя).
- **No-Root Mode:** Пошаговая помощь в очистке кэша Сервисов Google, Маркета и Диспетчера загрузок.
- **Интуитивный интерфейс:** Минимум текста, максимум дела.

### ❓ Почему Google ругается?
Приложение использует системные команды (для Root) и переходы в настройки. Антивирусы часто боятся таких действий, но код полностью открыт — вы можете проверить каждую строчку.
